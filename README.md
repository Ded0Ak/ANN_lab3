## Короткий опис
У цій лабораторній роботі реалізовано перцептрон в стилі OOP на основі TensorFlow Keras. Модель навчається одночасно вирішувати дві булеві операції: XOR та AND. Проєкт містить код для навчання, тестування, візуалізації процесу навчання та збереження/завантаження моделі.

## Зміст репозиторію
- `lab3.py` — основний скрипт з класом `XORAndPerceptron` та прикладом запуску (train / test / plot / save).
- `requirements.txt` — список залежностей для встановлення через pip.
- `xor_and_model.h5` — (опціонально) збережена модель після навчання.
- `training_history.png` — (опціонально) графік процесу навчання (генерується при наявності matplotlib).

## Вимоги
- Python 3.9–3.11 (рекомендується CPython)
- pip
- Рекомендовані пакети (див. `requirements.txt`):
  - tensorflow (або tensorflow-cpu)
  - numpy
  - matplotlib

Приклад `requirements.txt`:
```text
tensorflow-cpu>=2.15.0
numpy>=1.24.0
matplotlib>=3.7.0
```

## Встановлення
1. Встановіть Python і переконайтеся, що під час інсталяції обрано "Add Python to PATH".
2. Оновіть pip:
```bash
python -m pip install --upgrade pip
```
3. Встановіть залежності:
```bash
python -m pip install -r requirements.txt
```
Якщо команда `pip` недоступна — використовуйте `python -m pip`. На Windows при помилках завантаження TensorFlow (наприклад, `DLL load failed`) встановіть Microsoft Visual C++ Redistributable (https://aka.ms/vs/17/release/vc_redist.x64.exe) або використайте `tensorflow-cpu`.

## Запуск
Приклад запуску основного скрипта:
```bash
python lab3.py
```
Скрипт поетапно:
- створює модель,
- виводить архітектуру,
- навчає модель (за замовчуванням 500 епох),
- тестує на всіх чотирьох входах,
- намагається побудувати графіки навчання,
- зберігає модель у `xor_and_model.h5`.

## Формат вхідних і вихідних даних
- Вхід: пари бінарних значень [0, 0], [0, 1], [1, 0], [1, 1].
- Вихід: вектор з двох бінарних значень:
  - колонка 0: XOR
  - колонка 1: AND

## Очікувані результати
Модель має правильно передбачати обидві операції для всіх чотирьох комбінацій вхідних значень. У виводі тесту для кожної комбінації буде надруковано значення передбачення (в діапазоні [0,1] і округлене до 0/1) та очікуване значення з позначкою CORRECT/WRONG.

## Налагодження проблем
- Якщо при імпорті TensorFlow виникає помилка `DLL load failed while importing _pywrap_tensorflow_internal`, встановіть Microsoft Visual C++ Redistributable (посилання вище).
- На Windows часто стабільніше використовувати `tensorflow-cpu`.
- Якщо версія Python не сумісна з встановленою версією TensorFlow — спробуйте Python 3.10.
- Використовуйте `python -m pip install ...`, якщо `pip` недоступний у PATH.
